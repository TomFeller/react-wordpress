Link component

Component usage

The <Link /> component is use to render a basic 'a' tag.

Props

Prop name Description Is required Default value
label The text/content of the link Yes -
href The url of the link Yes javascript:void(0)
target The target type of the link. No -

Live example:

Copy that code:


    <Link label='Go'
          href='www.google.com'
          target='_blank' />

To render:


    <a href="www.google.com" target="_blank"> Go </a>
