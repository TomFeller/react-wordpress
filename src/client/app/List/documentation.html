List component

Component usage

The <List /> component is use to render a <ul> tag with <li> tags.

Props

Prop name Description Is required Default value
name Add as a class name for the <ul> tag and also as a BEM block class for the items (see example below). Yes -
items The items (<li>) that are inside the <ul> Yes -
horizontal if set to true, the <ul> will have 'list--horizontal' class (in addition to the 'list' class) No false
hasLinks Tell the component if the items should have <a> tag inside. If set tu 'true', the
<Link /> component will render. 		No false

Live example:

Copy that code:


  const items = [
      {
        label: 'home',
        href: 'home.html',
      },
      {
        label: 'about',
        href: 'about.html'
      },
      {
        label: 'contact',
        href: 'contact.html'
      }
  ];
  <List name='nav'
        items={items}
        horizontal
        hasLinks />

To render:


    <ul class="header-nav list list--horizontal">
        <li class="header-nav__item">
            <a href="home.html">home</a>
        </li>
        <li class="header-nav__item">
            <a href="about.html">about</a>
        </li>
        <li class="header-nav__item">
            <a href="contact.html">contact</a>
        </li>
    </ul>
