Image component

Component usage

The <Image /> component is use to render a basic 'img' tag.

Props

Prop name Description Is required Default value
src The source of the image Yes -
alt The alt of the image Yes -

Live example:

Copy that code:


    <Image src='images/example.png'
           alt='example' />

To render:


    <img src='images/example.png' alt='example' />
